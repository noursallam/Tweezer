Overview

Tweezer is a sophisticated Python utility engineered for the precise identification and extraction of sensitive information across various document formats and web resources. Our tool combines powerful pattern matching capabilities with an intuitive interface to provide comprehensive data analysis solutions.

Important Notice

Ensure compliance with applicable regulations and obtain proper authorization before scanning any systems or documents.

Key Features

Pattern Matching

Advanced algorithms for detecting over 20 types of sensitive information patterns.

Configurable Scanning

Highly customizable scanning options with support for multiple file types and URLs.

Real-time Analytics

Monitor scanning progress with detailed analytics and visual feedback.

Installation

Prerequisites

  • Python 3.8+
  • pip (Python package manager)

Installation Steps

git clone https://github.com/noursallam/Tweezer.git
cd Tweezer/tool
sudo chmod +x setup.sh
./setup.sh

The setup.sh script will:

  • Install required Python dependencies from requirements.txt.
  • Set up an alias for tweezer in your shell configuration file (.bashrc or .zshrc).
  • Make the Python scripts (twz.py, demo.py, flag.py) executable.
source ~/.bashrc # or source ~/.zshrc

Usage

File Scanning

tweezer [filename] -k [options]

URL Scanning

tweezer -y https://example.com -k [options]
